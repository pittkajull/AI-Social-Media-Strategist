## Cara Mengambil Screenshot

### n8n Workflow
1. Buka workflow di n8n
2. Pastikan semua node terlihat
3. Zoom ke level yang optimal
4. Screenshot dengan background putih (lebih bersih)

### Supabase Dashboard
1. Buka table yang ingin di-screenshot
2. Tampilkan beberapa baris data
3. Pastikan column headers terlihat jelas

### Threads Content
1. Buka post di Threads
2. Tampilkan engagement count
3. Screenshot dalam mode light mode (lebih clean)

---

## Format dan Resolusi

- Format: PNG (untuk kualitas terbaik)
- Resolusi minimal: 1920x1080
- Background: Putih atau light mode
- Annotation: Tambahkan panah atau highlight jika perlu

## Tools yang Direkomendasikan

- **Windows:** Snipping Tool, ShareX
- **Annotation:** Figma, Canva, atau langsung di image editor
